Rules of the Game !!

1. Roll the dice, the current value will be same as dice value.
2.You can roll dice untill you get the dice value of "1" .
3.Once you get the dice value of "1" ,current score you earned will be set to zero, then player control will be switched to another player.
4 The player who reaches 100 total point(final point) ,wins the game .
5. You can also roll the dice add the current value to final value by pressing "Hold button".Once you press the hold button current value will be added to the final value and player control will be switched.
